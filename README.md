## P2P Трекер (Server)
Трекер связок P2P (сервер).

**Требования**:

 1. Node JS >= 16
 2. Yarn

**Установка**:

 1. git clone https://github.com/OverTheGalaxy/P2P-Server.git
 2. cd P2P-Server
 3. yarn
 4. yarn serve
 5. Сервер будет запущен на 3500 порту
 6. **Первые данные на сервере появятся примерно спустя 30 секунд**
